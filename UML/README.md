# Adhan UML

UML Diagrams based on the Java implementation. These are to aid anyone who would like to convert 
it to other languages easily. 

## Usage

See [plantuml](http://plantuml.com) website for more information on plantuml


### Produce images
```
plantuml file.puml
```
